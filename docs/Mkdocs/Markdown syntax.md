This page includes all the basic markdown syntax to get started with creating a document

## Formatting text

### Headings

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
...
```

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

### Links

External links

```md
[Google.com](https://www.google.com/)
```

[Google.com](https://www.google.com/)

Internal links

```md
[[index]]
```

[[index]]
[[Callouts]]

### Images
